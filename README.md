## ğŸš€ Getting Started

### Prerequisites

- Node.js 16+ 
- npm or yarn package manager
- Git

### Development Setup

```bash
# Clone the repository
git clone https://github.com/your-username/Pulse.git
cd Pulse

# Install all dependencies
npm install

# Start Pulse Editor for development (with hot reload)
npm run dev

# Build production version
npm start

# Run tests
npm run test:editor
```
